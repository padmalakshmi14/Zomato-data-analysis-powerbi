# Zomato Restaurant Data Analysis (Power BI)

Project Overview

This project analyzes restaurant data from Zomato using Power BI. The goal of the dashboard is to explore restaurant ratings, cuisines, locations, and pricing patterns to understand customer preferences and restaurant performance.

The interactive dashboard provides insights into restaurant distribution, popular cuisines, average ratings, and cost trends.

Objectives

* Analyze restaurant ratings and popularity
* Identify the most common cuisines
* Compare restaurant distribution across cities
* Understand cost trends for different restaurants
* Visualize restaurant performance using interactive dashboards

 Tools Used

* Power BI
* Excel / CSV dataset
* Data visualization techniques

 Dataset Information

The dataset contains information about restaurants listed on Zomato, including:

* Restaurant Name
* City / Location
* Cuisine Type
* Average Cost for Two
* Rating
* Votes
* Online Delivery Availability

 Dashboard Features

The Power BI dashboard includes the following visualizations:

* Total number of restaurants
* Average restaurant ratings
* Most popular cuisines
* City-wise restaurant distribution
* Cost distribution analysis
* Rating comparison across restaurants

 Key Insights

* Some cities have significantly higher restaurant density.
* Certain cuisines appear more frequently and are more popular.
* Restaurants with higher ratings tend to receive more customer votes.
* Pricing varies widely depending on location and cuisine type.
